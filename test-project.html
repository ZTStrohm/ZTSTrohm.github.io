b. Calculate Correlation Coefficients¶

We are looking for how each parameter correlates with price.

In [7]:
corr_a = data.price.corr(data.sqft_living)
corr_b = data.price.corr(data.sqft_lot)
corr_c = data.price.corr(2020-data.yr_built)
corr_list = [corr_a,corr_b,corr_c]
d = {'Corr. Coef':corr_list}
d = pd.DataFrame(d,index=['Living Area','Lot Area','Age'])
d.head()
Out[7]:
Corr. Coef
Living Area 0.702035

We can see above that living area has the strongest correlation with price. Lot area is weakly positively correlated with price, while age is weakly negetively correlated with price.
